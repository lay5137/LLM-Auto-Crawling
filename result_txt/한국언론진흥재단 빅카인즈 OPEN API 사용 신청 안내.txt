한국언론진흥재단 빅카인즈 OPEN API 사용 신청 안내입니다. 관련 부서는 공주대학교 SW중심대학사업단이며, 작성일은 2025년 10월 24일입니다. 자세한 내용은 https://swknu.kongju.ac.kr/community/noticedetail.do?seq=134 에서 확인할 수 있습니다.

국립공주대학교 SW중심대학사업단과 한국언론진흥재단의 업무협약 체결이 완료됨에 따라 한국언론진흥재단의 빅카인즈 OPEN API 서비스 이용이 가능하게 되었으며, 사용 신청 절차를 안내드립니다. 먼저 빅카인즈 OPEN API 사용신청서(첨부파일)를 작성합니다. 다음으로 구글폼(https://forms.gle/zKMeu46fGYX5ZE286)을 작성해 제출하며, 이때 구글폼에 신청서 파일을 업로드합니다. 이후 SW중심대학사업단의 승인을 받습니다. 마지막으로 빅카인즈 OPEN API 사용 매뉴얼과 사용 키가 메일로 안내됩니다. 안내 메일은 구글폼에 기재한 메일로 발송될 예정이므로, 구글폼 작성 시 정확한 메일 주소를 기재해 주시기 바랍니다.

첨부파일은 다음과 같습니다. 빅카인즈 OPEN API 서비스 이용안내.jpg(https://swknu.kongju.ac.kr/boardfiledownload.do?seq=83)와 빅카인즈 OPEN API 사용 신청서.hwp(https://swknu.kongju.ac.kr/boardfiledownload.do?seq=84)입니다.
