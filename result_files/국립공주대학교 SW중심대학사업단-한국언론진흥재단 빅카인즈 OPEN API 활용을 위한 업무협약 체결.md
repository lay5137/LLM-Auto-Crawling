# 국립공주대학교 SW중심대학사업단-한국언론진흥재단 빅카인즈 OPEN API 활용을 위한 업무협약 체결

**작성자:** 관리자  
**작성일:** 2025-10-24  

---

## 본문
국립공주대학교(총장 임경호) SW중심대학사업단(단장 임재현)과 한국언론진흥재단(이사장 김효재)은 지난 10월 17일, 뉴스 데이터 플랫폼 ‘빅카인즈(BigKinds)’를 활용한 AI 데이터 분석 및 교육, 연구 협력을 위한 업무협약(MOU)을 체결했다.

이번 협약은 공공·교육기관의 데이터 기반 연구를 활성화하고, 뉴스 데이터와 인공지능 기술을 접목한 실증 서비스 및 교육 콘텐츠 개발을 추진하기 위해 마련됐다.

양 기관은 ▲빅카인즈 뉴스 데이터 제공 및 API 활용 지원 ▲뉴스 요약·분류 등 AI 응용기술 연구 협력 ▲데이터 기반 교육·실습 및 공동 연구 추진 등을 주요 협력 과제로 삼고 상호 발전을 위해 협력하기로 했다.

국립공주대학교 SW중심대학사업단은 이번 협약을 통해 뉴스 데이터를 기반으로 한 자연어처리, 정보 추천, 음성합성 등 실무형 연구를 확대하고, 학생들의 데이터 활용 역량을 강화할 계획이다.
