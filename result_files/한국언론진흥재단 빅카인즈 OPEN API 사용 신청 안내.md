## 본문
국립공주대학교 SW중심대학사업단과 한국언론진흥재단과의 업무협약 체결이 완료됨에 따라 한국언론진흥재단의 [ 빅카인즈 OPEN API ] 서비스 이용이 가능하게 되었음을 알려드리며, 사용 신청 절차 안내드립니다.


1) 빅카인즈 OPEN API 사용신청서(첨부파일) 작성

2) 구글폼 작성 및 제출
- 구글폼 링크: https://forms.gle/zKMeu46fGYX5ZE286
* 구글폼에 신청서 파일 업로드 *

3) SW중심대학사업단 승인

4) 빅카인즈 OPEN API 사용 매뉴얼 및 사용 키 메일로 안내
※ 해당 안내 메일은 구글폼에 기재한 메일로 발송될 예정이오니, 구글폼 작성시 정확한 메일주소 기재바랍니다.
